  1. shell commands:

  • ls
    • -F: show extra information like "/" at the end of a folder
  • man [function] - show help page for function
  • cd : change directory
  • pwd : print working directory 
  • nano : text editor for the command line
  • touch : makes a new blank file (or updates the modified time of an existing file!)
  • mv [input] [output] : moves file from input to output
    • -i : interactive, will ask you if you want to overwrite
